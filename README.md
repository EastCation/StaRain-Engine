# StaRain Engine/星雨引擎

一个基于Python的[Phigros](https://pigeon-games.com/phigros)模拟器

>*请不要与隔壁游戏星引擎Party混淆*

## 实现功能

已实现(Finished)：可游玩谱面

进行中(Processing)：将midi转为可读取谱面（已完成50%）

考虑中(Pending)：导入PEZ，曲目选择界面

## 先决条件

[Cocos2d(Python)](http://python.cocos2d.org/)

使用如下命令安装:
```shell script
pip install cocos2d
```

## 示例谱面

```shell script
python demo.py
```
